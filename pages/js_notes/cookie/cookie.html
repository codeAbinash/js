JavaScript Cookies

What is a cookie? How does it work? How to create a cookie in JavaScript?

Hey cookie is an amount of information the persist between a server side and client side. Every browser store this information at the time of browsing. A cookie contains the information as a string, generally in the form of a name-value pairs separated by semicolons. The state of a user and remembers the users information along all the web pages.

When a user sends request to the server, then each of their request is treated as a new request sent by the different users. So to recognize the old user, we need to add the cookie with the response from the server.

Now, whenever user sends request to the server the cookie is added with the request automatically. Due to the cookie the server recognizes the user.

In JavaScript which can create, read, update and delete a cookie by using document.cookie property. The following syntax is used to create a cookie. document.cookie = "name=vale". For example -

JavaScript provides some optional attributes that enhance the functionality of cookies.

Expire

It maintains the state of a cookie up to the specified date and time.

Max-age

It maintains the state of a cookie after a specified time. Here time is given in seconds.

Path

It extends the scope of the cookie to all the pages of a website.

Domain

It is used to specify the domains for which cookies valid.

JavaScript Event

What do you mean by event? What is event handler? Explain the various event handlers in JavaScript.

An event is an action that occurs in the web browser, which the web browser feedbacks to the user so that user can respond to it.

For example, when user clicks a button on a web page, he or she may want to respond a click event by displaying a dialogue box.

Each event may have an event handler which is a block of code that will execute when the event occurs. And event handler is also known as an event listener. It listens to the event and executes when the event occurs.

Suppose we have a button with ID btn

To define the code that will be executed when the button is clicked, we need to register an event handler using the addEventListener() method.

* First select the button with the id btn using the getElementById() method.

* Then, define a function called display as an event handler.

Finally register an event handler using the addEventListener() so that when user clicks the button display function will be executed.

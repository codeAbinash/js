JavaScript is an object oriented programming language. A programming language that can be called object oriented if it provides 4 basic capabilities.

The eligibility to store related information, whatever data or methods, together in an object.

That capability to store one object inside object.

3. Inheritance

The capability of a class to relay upon another class or member of classes for some of its properties and methods.

The capability to write one function all method that walks in beauty of Wales.

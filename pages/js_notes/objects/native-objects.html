JavaScript Native Objects

JavaScript has many built-in native objects. These objects are accessible anywhere in the program and will work.The same way in.All browsers running in any operating system.

The important JavaScript Native objects are -

  1. JS Number Object
  2. JS Boolean Object
  3. JS String Object
  4. JS Array Object
  5. JS Date Object
  6. JS Math Object
  7. JS REgexp Object

The Number Object

The number of jet represents numerical data, either integers or floating point numbers. The sentence for creating a number of justices follows.

In the place of number, if we provide any non number argument then if the argument cannot be converted into a number, it returns NaN (Not a Number).

The properties associated with number object are Max_VALUE, Min_VALUE, NaN, NEGATIVE_INFINITY, POSITIVE_INFINITY, prototype, constructor etc.

The methods associated with number object are - toExponential(), toFixed(), toLocaleString(), toPrecision(), toString(), valueOf() etc.

The Date object

the Date object is a data type built into the javascript language. Date objects are created with the new Date() as follows. Once a Date object is created, a number of methods allow us to operate on it. Most methods simply allows us to get and set the year, month, day, hour, minute, second and milliseconds fields of the object, using either local time or UTC(universal or GMT) time. The ECMAscript standard the Date object to be able to represent any date and time, to millisecond precision, with in 100 million days before or after 01/01/1970. This is a range of plus or minus 273785 years, so javascript run represent date and time till the year 275755.

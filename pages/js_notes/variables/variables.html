What do you mean by local variable and global variable in JavaScript.

Variable hold the data or information which can be changed anytime. JavaScript use keyword var these are variables. In JavaScript there are two types of variables

Local Variable

Local variables are the variables defined inside the functions.They have local scope, which means that they can only be used within the functions that define them. Since local variables are defined inside the functions so variables with the same name can be used in different functions.
